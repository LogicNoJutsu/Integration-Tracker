⚙ GitHub Sync Setup
Your data will be saved as data.json in your GitHub repository every time you make a change.

How to get a Personal Access Token:
1. Go to github.com/settings/tokens/new
2. Give it a name (e.g. "CRx Tracker")
3. Check repo scope (full repository access)
4. Click "Generate token" and copy it here
